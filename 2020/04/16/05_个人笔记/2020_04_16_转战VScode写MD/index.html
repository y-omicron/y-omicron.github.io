转战VScode写MarkDown文档

Posted on 2020-04-16 Edited on 2020-04-17 In 教程

以前使用的都是Typora，但是它太过于强大，导致我导出的文章放到博客上或多或少的会出现BUG，真是难受…

修改Post模板

Path

  • 路径很简单，我们找到路径<you blog>/scaffolds/这里存放的都是模板文件，我选择新建一个模板，因为我喜欢加密一部分文章，所以我创建了一个encode.md，我是这样编辑我的模板的。

下载VS Code插件

图片处理

  • 什么都不要说，先容我解决粘贴图片的问题！！！这也是为什么我一直使用的是Typora的原因，但是我发现VS Code有插件Paste Image可以解决这个问题
  • 然后需要简单的配置一下，将复制黏贴的图片放到指定的文件夹中<DirPath>\<FileNameDir>
  • ${currentFileDir}\${currentFileNameWithoutExt}
  • 按照文档的说法，这两个变量是当前文件路径和当前文件不带Ext的名称

预览插件

  • 没什么好说的，大名鼎鼎的Markdown Preview Enhanced，下载好之后，可以在页面的右上角点击对应的预览，来使用这个插件渲染你的MD文档

快捷操作

  • 这个主要是Markdown All In One提供的快捷键让人爱不释手，所以也下载安装了一手，具体的快捷键可以前往GitHub主页查看~
  • markdown all in one 的 GitHub 主页

建立文件
