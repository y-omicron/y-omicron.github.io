主要是弄来弄去，搭建N次虚拟机，总是忘记东西，索性记录下来

源

  • 没什么好说的，直接上链接
    • Ubuntu（网易镜像源）
    • VMware 15 Pro（含注册码）
  • 关于源，特殊的一点就是一般在搭建好虚拟机之后需要更换原本的源(/etc/apt/sources.list)为国内的镜像源用来提高速度，这个文件呢，总是要去网上找，然后有的人又坑，复制出来的自己的源总会缺失一部分，导致出现各种各样的错误，所以我更换源的时候，直接将原来源文件中的cn.archive.ubuntu替换为mirrors.163，顺便附上我使用的sources.list文件

虚拟机设置

设置中文

  • 首先说一下中文这个事情
    1. 可以在创建虚拟机的时候选择语言为中文，但是不建议，因为会有一部分文件夹的名字变成中文，某些脚本之类的会与中文路径冲突
    2. 修改语言为中文之前必须apt-get update，没有用管理员权限执行这条命令的也自己了断吧
    3. 更换好了之后系统依然没有中文输入法，请各位各显神通，这里提供搜狗输入法的链接

设置系统语言为中文

  • 如何设置中文，首先终端执行apt-get update，有强迫症的可以顺便执行apt-get upgrage
  • 然后我们打开系统设置，就是右上角那个齿轮里面的System Settings（这俩单词不认识的建议退出计算机相关工作）
  • 接下来找到语言设置，第一次点进去的时候会下载一些东西，点击下载就行了(Install)
  • 接下来添加或删除语言，在里面勾选简体中文，然后点击应用
  • 然后的事情就很诡异简单，将简体中文拖到最上面就可以了
  • 接下来直接reboot，说句题外话，我在上一步中遇到很无语的情况，无法拖动简体中文，显示为灰色，最终通过修改配置文件完成修改的操作，修改方式如下：
    1. 找到Ubuntu语言的配置文件
      gedit /etc/default/locale
    2. 修改文件为:
    3. 重启之后就可以修改系统的语言了，其中LANGUAGE是登录界面的语言，可加可不加
  • 接下来重启之后会询问你是否更换名称，千万不要点击是，不然这白干了，就是为了保留旧名称(英文)，同时利用中文化的系统来帮助我们阅读设置

安装中文输入法

  • 我就不用搜狗输入法了，不是很方便，直接使用默认的ibus-pinyin输入法，虽然现在已经更改系统语言为中文了，但是依然没有输入法，首先下载输入法
    sudo apt-get install ibus-pinyin
  • 然后ibus-setup打开ibus设置，添加中文输入法
  • 接下来重启之后就可以看到拼音输入法已经安装好了~
