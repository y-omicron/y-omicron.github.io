第一部分：key == 2

  • 其实上面就知道了，key == 2 对应的是下，但是我们这里还是要好好分析一下
  • 当 key == 2 的时候v4++
  • 分析一下，首先是 *(&v4 + i) < 0 || *(&v4 + i) > 4 对应的是退出程序
  • 第一点，i == 0恒成立…很不可思议…
  • 那么初始v4的值为0，当InPut == 2的时候，v4也是自加一，那么必然不会是小于0的，因为她必然等于1，就很脑残…
  • 这里解释一下如何计算的，首先是&符号：取地址符，表示取出这个变量的地址，然后是*符号：取值符，表示取出这个地址里面的数值…
  • 嗯…没错，这段代码就是在判断v4 < 0 || v4 > 4，可以算是一种干扰分析的手法吧
  • 接下来，*((_BYTE *)&v8 + 5 * v4 + v5 - 0x29) == '1'这句代码非常重要，分析我就放下面了，哈哈哈~

第二部分：key == 3

  • 嗯，作用就是v5++，其他的留给后面分析 QAQ

第三部分：key != 4

  • 你想想，大于2之后，等于3已经被取走了，不等于4的是不是就就大于4，是不是不满足条件了，所以直接退出，排除大于4的值
  • 否则++v5也就是key == 4的时候++v5

第四部分：key != 1

  • 一样的道理，大于2的已经都判断了，这个时候不等于1，必然是小于等于0的，不满足条件，直接退出
  • key == 1的时候--v4
