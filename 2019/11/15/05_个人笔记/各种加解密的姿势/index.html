AES_256_CBC

  • 函数原型：

    • const unsigned char *in：待加密的数据（我更习惯使用BYTE[]类型）。
    • unsigned char *out：输出；
    • size_t length：加密长度，注意控制长度，最好分段加密
    • const AES_KEY *key：AES密钥，前面使用set_aes_key获得
    • unsigned char *ivec：Iv，偏移量，使用之后，这个值会发生改变，连续使用记得修改。
    • const int enc：模式；AES_ENCRYPT 或者AES_DECRYPT
